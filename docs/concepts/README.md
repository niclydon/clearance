## The Core Idea

Clearance treats work management as an operating system for a live technical environment.

The PMO layer should answer four questions at any time:

1. What work exists?
2. What work is real, proposed, blocked, ready, or done?
3. Who or what is allowed to act on each item?
4. What evidence proves the outcome?

The answer should be inspectable without reading private chat history or reconstructing context from memory.
